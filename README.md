# Anil-Calculator

A very simple calculator built with **Kotlin**, supporting basic arithmetic operations.

## Features
- Perform basic math operations: `+`, `-`, `*`, `/`
- Lightweight and fast
- Simple and user-friendly interface
- **Dark mode support**
